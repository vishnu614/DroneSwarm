DroneSwarm is a full-stack project designed to **manage and control a swarm of drones** for various tasks such as monitoring, mapping, or delivery. The project provides a backend API, a frontend interface, and simulation/control scripts for drone coordination.

---

## Features

- **Multi-Drone Coordination:** Control multiple drones simultaneously.
- **Task Assignment:** Assign tasks to drones dynamically.
- **REST API Backend:** Easily extendable for new functionalities.
- **Frontend Dashboard:** Visualize swarm operations.

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Flask 
- **Drone Communication:** WebSocket / REST API
